
## Descrição

Encutador Simples de URL

## Setup do Projeto

Crie e preencha o arquivo .env

```bash
$ yarn install
```


## Compilar and rodar o projeto

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Rodar testes

```bash
# unit tests
$ yarn run test

```

## Docker

Defina a env DB_URL = db

```bash
$ docker build -t app .

$ docker compose up -d 
```
